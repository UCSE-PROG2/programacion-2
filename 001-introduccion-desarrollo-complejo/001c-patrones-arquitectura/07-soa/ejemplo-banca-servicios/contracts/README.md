# Contratos / API (`contracts/`)

Definición de endpoints, modelos y códigos de error para cada servicio, para que los consumidores sepan cómo llamarlos.

## Archivos que deberían ir en esta carpeta

- **openapi.yaml** (o varios archivos por servicio) — Definición OpenAPI de los endpoints de autenticación, cuentas y transferencias, con modelos (DTOs) y respuestas de error.
